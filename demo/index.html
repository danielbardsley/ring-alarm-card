‚ú® Control Buttons Features

  • Three Action Buttons: Disarmed, Home, and Away for quick state changes
  • Visual State Indication: Active button is highlighted with appropriate color
  • Transitional State Handling: Buttons disabled during arming/disarming/pending/triggered states
  • Progress Indicator: Circular border progress fills during countdown periods
  • Countdown Timing: Progress syncs with exitSecondsLeft attribute from alarm entity
  • Loading States: Visual feedback when a service call is in progress
  • Error Handling: Error indication with automatic recovery after 3 seconds
  • Accessibility: Full ARIA support with labels, pressed, disabled attributes, and live region announcements
  • Reduced Motion: Respects prefers-reduced-motion setting for animations
  • Theme Integration: Uses Home Assistant CSS custom properties for consistent styling
  • Compact Mode: Smaller button size for space-constrained dashboards
